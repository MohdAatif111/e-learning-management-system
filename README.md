# E-Learning Management System

## Project Overview
E-Learning Management System is a web-based application developed using Python, Flask, HTML, CSS and SQLite. The system helps manage students, courses and enrollments in an online learning environment.

## Features
- Student Login
- Course Management
- Student Enrollment
- Learning Dashboard
- Database Integration

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite
- SQL
